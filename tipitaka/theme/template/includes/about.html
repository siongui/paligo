Welcome to Online Pāḷi Tipiṭaka Website!

This website (source code, data) allows instant lookup of words when mouse cursor hovers over words, and contrast (parallel) reading of Pāḷi texts and translations (if available). The dictionaries include Pāḷi-English, Pāḷi-Japanese, Pāḷi-Chinese, Pāli-Vietnamese, Pāli-Burmese. The data of Pāḷi Tipiṭaka in this website comes from Pāḷi Tipiṭaka. Before using this website, please read Howto on Pāḷi Tipiṭaka first. The tipiṭaka here is Burmese edition. For Thai, Sri Lankan, European PTS editions, visit Pāli Texts Collection. If you only need to look up Pāḷi words, please visit Pāḷi Dictionary. The data of dictionaries comes from Pali Canon E-Dictionary Version 1.94 (PCED). This website is still under development. If any questions, problems, or suggestions, please contact me.
